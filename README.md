# Aula_Genetica_Estatistica

## Propósito 
Meu propósito principal é aprender e aplicar ferramentas estatísticas voltadas para análises de dados em genética e biotecnologia.

## Conteúdo Futuro
O foco das próximas atualizações deste repositório será a consolidação de fluxos de trabalho 
estatísticos que integram a minha atuação profissional no Centro de Tecnologia Canavieira (CTC) 
com a pesquisa acadêmica de mestrado. Pretendo desenvolver scripts que automatizem o 
processamento de dados de experimentos de rotina em biotecnologia, permitindo uma análise 
mais ágil e precisa de ensaios laboratoriais. 
Além disso, o espaço será utilizado para análises que irei fazer durante o mestrado, analisando o efeito de mutante da bactéria *Pantoea* nas respostas biologicas do microrganismo e também de interação com planta.  
O objetivo final é criar um ambiente de análise reprodutível, onde o conhecimento 
adquirido durante todo esse processo possa ser utilizado em análises futuras que vierem ocorrer.
